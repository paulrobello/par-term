par-term

A cross-platform GPU-accelerated terminal emulator

Quick Install

Install shaders (50+ custom background and cursor shaders):

$ curl -sSL https://paulrobello.github.io/par-term/install-shaders.sh | sh

Install shell integration (directory tracking, command notifications, CWD sync):

$ curl -sSL https://paulrobello.github.io/par-term/install-shell-integration.sh | sh
